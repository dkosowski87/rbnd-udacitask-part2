# What you added to Udacitask 2.0
# -------------------------------------------------------
1. What is the gem of your choice and how did you use it?
	I used the terminal-table gem to create a better looking display of UdaciLists in the command line. 
2. Describe the 2 features you added to the project.
  a) Users - the possibility to create users and scope lists to particular users. A user must have an unique name and a valid password. I user can have many lists.
  b) Interface - I used the HighLine gem to create an interface for the app. Now the user can register/login for the app and create lists, view lists, add items, etc. through the command line interface.
